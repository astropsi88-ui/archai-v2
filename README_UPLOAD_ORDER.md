# ArchAI — пакет для GitHub

Структура:

- `docs/content/` — смысловые документы и финальные тексты страниц.
- `docs/codex/` — задания для Codex.

Порядок работы:

1. Выполнить только `docs/codex/01_SHARED_FOUNDATION.md`.
2. Проверить результат.
3. Выполнить только `docs/codex/02_MAIN_PAGE.md`.
4. Проверить результат.
5. Открыть `docs/codex/03_PRODUCT_PAGES_RUNBOOK.md` и выполнять этапы по одному:
   - 03A Компаньон
   - 03B Напарник
   - 03C Соавтор
   - 03D Личность проекта
   - 03E Сайты с AI
   - 03F финальный QA

Не давать Codex все этапы одним запросом.
